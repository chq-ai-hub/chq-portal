- 响应式布局（适配桌面端和平板）
- GitHub Actions 自动部署配置
- 完整的 README 和集成指南文档

### 设计
- 苹果级极简美学（SF Pro 字体栈、大圆角、留白）
- 欧莱雅高级感配色（暖金、玫瑰金强调色）
- 60fps 流畅交互动画

## 版本命名规范

| 类型 | 示例 | 说明 |
|------|------|------|
| 主版本 | v1.0.0 → v2.0.0 | 重大架构变更或不兼容更新 |
| 次版本 | v1.0.0 → v1.1.0 | 新增功能、新页面集成 |
| 修订版本 | v1.0.0 → v1.0.1 | Bug 修复、样式微调 |

### 文件命名规范

页面文件：`{名称}_v{版本号}.html`
示例：`LLM-对比平台_v1.2.3.html`

---

## 历史版本备份

每次发版时，请将上一个版本的完整工程压缩包放入 `versions/` 目录，命名格式：
`chq-portal-{版本号}-{日期}.zip`
示例：`chq-portal-v0.9.0-20260715.zip`
