# versions/ 目录说明

此目录用于存放历史版本的完整工程备份。

## 备份命名格式

```
chq-portal-v{版本号}-{日期}.zip
```

**示例**：
- `chq-portal-v1.0.0-20260716.zip`
- `chq-portal-v1.1.0-20260801.zip`
- `chq-portal-v2.0.0-20260915.zip`

## 何时需要备份

- 发布重大版本更新前
- 进行不兼容的结构调整前
- 需要保留某个稳定版本的快照时

## 备份内容

备份应包含工程的完整内容：
- 所有 HTML、CSS、JS 文件
- pages/ 目录下的所有页面
- pages.json 配置文件
- 文档和说明文件
